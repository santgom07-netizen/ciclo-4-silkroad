Este proyecto de DOPO es un simulador innovador desarrollado en Java que recrea de manera detallada la histórica Ruta de la Seda, integrando robots autónomos y tiendas virtuales para simular las dinámicas reales de comercio, transporte e intercambio de mercancías que caracterizaron esta red comercial fundamental entre Oriente y Occidente. El simulador permite a los usuarios experimentar con algoritmos complejos, sistemas de enrutamiento, gestión de inventarios y lógica de negociación comercial, proporcionando un entorno educativo interactivo donde se pueden explorar decisiones estratégicas, visualizar cómo los robots navegan por las rutas comerciales y cómo las tiendas gestionan sus transacciones. Este proyecto está enfocado en fortalecer habilidades de programación orientada a objetos, modelado de sistemas, pensamiento computacional y comprensión de procesos logísticos y económicos, siendo una excelente herramienta pedagógica para aprender sobre algoritmos de optimización, simulación y estructuras de datos complejas en un contexto histórico y comercial fascinante.
